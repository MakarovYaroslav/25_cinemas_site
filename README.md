# Cinemas Site

This [site](https://cinemassite.herokuapp.com/) displays list of popular films with reference information.

# Deploy on localhost

Example of launch on Linux, Python 3.5:

```bash
gunicorn server:app
```

Open page [localhost:8000](http://localhost:8000) in browser.
